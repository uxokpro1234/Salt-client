# About

Salt is a custom Minecraft Forge 1.8.9 utility client/framework written entirely in Java.

The project started as an experimental client and was later completely rewritten with:
- a modular architecture
- custom setting system
- event-driven module handling
- ClickGUI framework
- rendering utilities
- Forge + Mixins integration

This project is mainly focused on:
- rendering systems
- Forge internals
- movement mechanics
- GUI development

# Technologies Used

- Java
- Minecraft Forge 1.8.9
- Sponge Mixins
- LWJGL/OpenGL
- Gradle
